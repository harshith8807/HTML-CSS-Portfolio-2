Real-time Satellite Tracking and Switching Algorithm

  • Developed an algorithm to track low earth orbit satellites above the horizon and switch between these satellites to facilitate continuous connectivity and data transfer.
  • Uses GPS module to determine receiver antenna location and identifies LEO satellites above the horizon using an extensive database. connection is made, and when the algorithm senses disruption, it immediately switches to the next closest satellite, maintaining a low link budget for efficient data link formation.
